Ok, what should I start from?

Find the index of start a substring or single character in another string with string.indexOf() method. Do it two times - for left and right markers. 

let start = line.indexOf(left);
let end =  line.indexOf(right);

Ok, how to find the text between those markers?

Use string.slice() method to get substring. Note, you should add the length of left marker (in this case 1) and the end position of slice is not included. It's exactly what you need. 

line.slice(start + 1, end);

So, how my final solution should look like?

You just need to wrap your code into function expression and return the substring you've got.

Here is the whole solution:

function betweenMarkers(line: string, left: string, right: string): string {

    let start = line.indexOf(left);
    let end =  line.indexOf(right);
    
    return line.slice(start + 1, end);
}

This solution uses regular expression to find specific pattern inside line, and ternary operator. 

function betweenMarkers(line: string, left: string, right: string): string {
    
    let result = line.match(new RegExp(`\\${left}(.*?)\\${right}`));

    return result ? result[1] : '';
}
