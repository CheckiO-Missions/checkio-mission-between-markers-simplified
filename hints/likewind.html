def between_markers(text: str, begin: str, end: str) -> str:
    if begin in text:
        begin_index = text.find(begin) + len(begin)
    else:
        begin_index = 0
        
    if end in text:
        end_index = text.find(end)
    else:
        end_index = len(text)
        
    return text[begin_index:end_index]
First of all, we're checking for the left marker in the text. If there isn't one, then the answer will start from the very beginning of the input string. If there is one, then the answer will start with the character following the last character of the marker. After that we're checking for the right marker in the string. If there is one, we're making it the limit for our input string. If the input text doesn't contain the right marker, then the end of the input text will be the limit of our answer.
