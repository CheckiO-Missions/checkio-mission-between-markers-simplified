Тобі дано рядок і два маркери (початковий маркер та кінцевий маркер). Тобі потрібно знайти текст, який заключений у ці два маркера. Але є декілька важливих умов.

Це спрощена версія місії Between Markers.

  • Початковий та кінцевий маркери завжди різні.
  • Початковий та кінцевий маркери завжди розміром в один символ.
  • Початковий та кінцевий маркери завжди є в рядку і ідуть один за другим.

Вхідні дані: Три аргумента. Всі три - рядок. Другий і третій агрументи - початковий та кінцевий маркери.

Вихідні дані: Рядок.

Приклад:

{% if interpreter.slug == "js-node" %} 
betweenMarkers('What is >apple<', '>', '<') == 'apple'
betweenMarkers('What is [apple]', '[', ']') == 'apple'
betweenMarkers('What is ><', '>', '<') == ''
betweenMarkers('[an apple]', '[', ']') == 'an apple'
{% else %} 
between_markers('What is >apple<', '>', '<') == 'apple'
{% endif %}

Як це використовується: Для парсінгу тексту.

Передумови: Не може бути більше одного початкового або кінцевого маркеру.
