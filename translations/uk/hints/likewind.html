Вам потрібно знайти індекс лівого маркера (якщо він є), правого (ознову ж - якщо він є) і повернути текст, який знаходиться між ними.
Как мне найти индексы этих маркеров?
Используйте метод str.find()
Чи могли б навести приклад коду?
Звісно. Але спершу, зверни увагу, що за допомогою 'find' ви знайдете індекс першого маркера. Для того, щоб дізнатись індекс першого символу тексту маркерами, необхідно добавити довжину лівого маркера. 
def between_markers(text: str, begin: str, end: str) -> str:
    begin_index = text.find(begin) + len(begin)
    end_index = text.find(end)
Як отримати текст, між маркерами?
Використуючи зріз (slice): 
        
def between_markers(text: str, begin: str, end: str) -> str:
    begin_index = text.find(begin) + len(begin)
    end_index = text.find(end)
    return text[begin_index:end_index]
begin_index - індекс першого символу тексту між маркерами. end_index - індекс першого символа правого маркера. Так як при зрізі символ, який знаходиться на позиції end_index не включаїться в результат - ми отримаємо потрібний нам текст. Іншими словами, останій символ, який фактично включається в нашу відповідь - символ на позиції 'end_index - 1', що як раз і є останім символом тексту між маркерами.
Що робити, якщо якогось маркера немає в стрічці?
Тоді необхідно почати відлік з с 0-ї позиції (якщо немає лівого маркера - begin) або закінчити останім символом вихідного тексту (якщо немає правого маркера - end). В випадку, якщо маркери відсутні - ви повині повернути всю стрічку.
Чи могли б ви пояснити детальніше?
Звісно. 
def between_markers(text: str, begin: str, end: str) -> str:
    if begin in text:
        begin_index = text.find(begin) + len(begin)
    else:
        begin_index = 0
        
    if end in text:
        end_index = text.find(end)
    else:
        end_index = len(text)
        
    return text[begin_index:end_index]
Сперва мы проверяем, чи є лівий маркер в тексті. Якщо його там немає - то відповідь буде починатися з самого початку          вхідного рядка. Якщо він там є, то відповідь буде починатися з символу, наступного за останнім символом маркера.          Потім ми перевіряємо наявність в рядку правого маркера. Якщо він там є - ми обмежуємо по ньому наш вихідний рядок.          Якщо правого маркера немає в вхідному тексті - обмежувачем нашої відповіді буде кінець вихідного тексту.
