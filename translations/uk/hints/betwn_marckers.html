Зверни свою увагу на str.index метод. Якщо ти відкриєш свою python консоль та впишеш txt = "Hello, welcome.", а потім виконаєш наступне txt.index("w"), то ти отримаєш у відповідь 7, тому що літера "w" знаходиться на 7-й позиції у рядку (рахувати починаємо з "0").

Добре. Як мені використати це для вирішення місії?

Твій початковий код має наступний вигляд: 

def between_markers(text:str, begin:str, end:str) -> str:
    # your code here
    return None

де text твоя перша змінна, яка являє собою рядок. Друга та третя змінна - початковий та кінцевий маркери. Отже, просто зроби наступне: 

text.index(begin)
text.index(end)

Хм, окей. Але як все ж таки повинен виглядати мій фінальний код? 

def between_markers(text: str, begin: str, end: str) -> int:

    return text[text.index(begin)+1:text.index(end)]
