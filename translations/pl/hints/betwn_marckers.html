Ok. Jak mogę użyć tego w rozwiązaniu?

Pana kod startowy powinien wyglądać następująco 

def between_markers(text:str, begin:str, end:str) -> str:
    # your code here
    return ""

co oznacza, że jako wartość zmiennej text mają Państwo ciąg znaków. Druga i trzecia zmienna to znaczniki początkowy i końcowy. Proszę więc wykonać następujące czynności... 

text.index(begin)
text.index(end)

Ostatnie pytanie. Jak będzie wyglądało moje ostateczne rozwiązanie? 

def between_markers(text: str, begin: str, end: str) -> int:

    return text[text.index(begin)+1:text.index(end)]
