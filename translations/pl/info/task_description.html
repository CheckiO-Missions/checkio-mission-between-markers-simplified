Dany jest string oraz dwa znaczniki: początkowy i końcowy. Twoim zadaniem jest znalezienie substringa zawartego pomiędzy tymi dwoma znacznikami.

To jest uproszczona wersja misji Between Markers.

  • Znacznik początkowy i końcowy różnią się.
  • Znacznik początkowy i końcowy są długości jednego znaku.
  • Znacznik początkowy i końcowy zawsze występują w stringu w kolejności końcowy po początkowym.

Wejście: Trzy argumenty. Wszystkie są typu string. Drugi i trzeci argument to znacznik początkowy i końcowy.

Wyjście: String.

Przykład:

{% if interpreter.slug == "js-node" %} 
betweenMarkers('What is >apple<', '>', '<') == 'apple'
betweenMarkers('What is [apple]', '[', ']') == 'apple'
betweenMarkers('What is ><', '>', '<') == ''
betweenMarkers('[an apple]', '[', ']') == 'an apple'
{% else %} 
between_markers('What is >apple<', '>', '<') == 'apple'
{% endif %}

Do czego jest używane: parsowanie tekstu.

Warunek wstępny: W tekście nie może znajdować się więcej niż jeden znacznik początkowy i jeden końcowy.
