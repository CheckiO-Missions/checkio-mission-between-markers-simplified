文字列と二つのマーカー（開始と終端）が与えられます。それらのマーカーの間にある部分文字列を返してください。条件は以下の通りです：

  • 開始マーカーと終端マーカーは必ず異なります。
  • 開始マーカーが存在しないときは、文字列の最初が開始地点です。
  • 終端マーカーが存在しないときは、文字列の最後が終端です。
  • 開始マーカーも終端マーカーも存在しないときは、文字列全体を返します。
  • 終端マーカーが開始マーカーより前にあるときは、空文字列を返します。

入力: 三つの引数で、すべて文字列。二つ目と三つ目が開始マーカーと終端マーカーを表す。

出力: 文字列。

例:

{% if interpreter.slug == "js-node" %} 
betweenMarkers('What is >apple<', '>', '<') == 'apple'
betweenMarkers('No[/b] hi', '[b]', '[/b]') == 'No'
{% else %} 
between_markers('What is >apple<', '>', '<') == 'apple'
between_markers('No[/b] hi', '[b]', '[/b]') == 'No'
{% endif %}

どのように使われるか: 文章のパース

前提条件: 開始マーカーは二つ以上存在しない。終端マーカーは二つ以上存在しない。
