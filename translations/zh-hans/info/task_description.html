你将得到一个字符串和两个标记（初始标记和最终标记）。你必须在这两个标记之间找到一个子串。但是有几个重要的条件。

  • 初始标记和最终标记总是不同的。
  • 初始标记和最终标记总是1个字符大小。
  • 初始标记和最终标记总是存在于一个字符串中，并且是一个接一个的。

输入：三个参数。它们都是字符串{% if is_js %}(string) {% else %} (str) {% endif %} 。第二个和第三个参数是初始和最终标记。

输出：一个字符串{% if is_js %}(string) {% else %} (str) {% endif %} 。

例子： 

{{init_code_tmpl}}

它是如何被使用的： 用于文本解析。

前提条件： 不能有多于一个最终标记和一个初始标记。
