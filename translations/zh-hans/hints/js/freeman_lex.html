好吧，我应该从什么开始呢？

用string.indexOf() 方法找到子串或另一个字符串中单个字符的起始索引。这样做两次--对左标记和右标记。 

let start = line.indexOf(left);
let end =  line.indexOf(right);

好的，如何找到这些标记之间的文本？

使用string.slice() 方法来获取子串。注意，你应该加上左边标记的长度（在本例中为1），并且不包括切片的结束位置。这正是你需要的。 

line.slice(start + 1, end);

那么，我的最终解决方案应该是什么样子的呢？

你只需要将你的代码包装成函数表达式，并返回你得到的子串。

下面是整个解决方案：

function betweenMarkers(line: string, left: string, right: string): string {

    let start = line.indexOf(left);
    let end =  line.indexOf(right);
    
    return line.slice(start + 1, end);
}

你把这一行分成两部分--左边的标记（取后半部分），然后右边的标记（取前半部分）。另外，在这个解决方案中使用了匿名函数。 

var betweenMarkers = (s: string, l: string, r: string): string =>
s.split(l)[1].split(r)[0];

这个解决方案使用正则表达式来寻找行内的特定模式，并使用三元操作符。 

function betweenMarkers(line: string, left: string, right: string): string {
    
    let result = line.match(new RegExp(`\\${left}(.*?)\\${right}`));

    return result? result[1]: '';
}
