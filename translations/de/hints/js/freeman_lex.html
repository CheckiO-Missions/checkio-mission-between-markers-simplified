💡 Sie müssen zwei Indizes (der linken und der rechten Markierung) und die Teilzeichenkette zwischen ihnen finden.

Ok, womit soll ich anfangen?

Finden Sie den Index für den Beginn einer Teilzeichenkette oder eines einzelnen Zeichens in einer anderen Zeichenkette mit der Methode string.indexOf(). Machen Sie das zweimal - für die linke und die rechte Markierung. 

let start = line.indexOf(left);
let end =  line.indexOf(right);

Ok, wie findet man den Text zwischen diesen Markierungen?

Verwenden Sie die Methode string.slice(), um eine Teilzeichenkette zu erhalten. Beachten Sie, dass Sie die Länge der linken Markierung (in diesem Fall 1) hinzufügen müssen und die Endposition des Slice nicht berücksichtigt wird. Das ist genau das, was Sie brauchen. 

line.slice(start + 1, end);

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck einpacken und die erhaltene Teilzeichenkette zurückgeben.

Hier ist die gesamte Lösung:

function betweenMarkers(line: string, left: string, right: string): string {

    let start = line.indexOf(left);
    let end =  line.indexOf(right);
    
    return line.slice(start + 1, end);
}

Sie teilen die Zeile zweimal auf - durch die linke Markierung (und nehmen die zweite Hälfte), dann durch die rechte Markierung (und nehmen die erste Hälfte). Außerdem wird in dieser Lösung eine anonyme Funktion verwendet. 

var betweenMarkers = (s: string, l: string, r: string): string =>
s.split(l)[1].split(r)[0];

Diese Lösung verwendet einen regulären Ausdruck, um ein bestimmtes Muster in der Zeile zu finden, sowie den ternären Operator. 

function betweenMarkers(line: string, left: string, right: string): string {
    
    let result = line.match(new RegExp(`\\${left}(.*?)\\${right}`));

    return result? result[1]: '';
}
