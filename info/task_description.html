You are given a string and two markers (the initial and final). You have to find a substring enclosed between these two markers. But there are a few important conditions:

This is a simplified version of mission Between Markers

  • The initial and final markers are always different.
  • The initial and final markers are always 1 char size.
  • The initial and final markers are always exist and go one after another.

Input: Three arguments. All of them are strings. The second and third arguments are the initial and final markers.

Output: A string.

Example:

{% if interpreter.slug == "js-node" %} 
betweenMarkers('What is >apple<', '>', '<') == 'apple'
{% else %} 
between_markers('What is >apple<', '>', '<') == 'apple'
{% endif %}

How it is used: for parsing texts

Precondition: can't be more than one final marker and can't be more than one initial
